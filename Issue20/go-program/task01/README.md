### 测试代码：

    package main
    import "fmt"
    func main()  {
	    fmt.Println("Hello World!!!")
    }

### 说明

* `package main` 表示一个可独立执行的程序，每个 `Go` 应用程序都包含一个名为 `main` 的包。

* `{}` 中"{"不可以单独放一行。
