### softmax

softmax 可以将神经元的输出映射到(0, 1) 区间，且和为一，这样就可以直接使用概率的方式来解释输出。

最大的特点是，求导计算很方便。尤其是跟交叉熵损失函数结合起来，求得的梯度很简洁。

即最小化交叉熵损失函数等价于最大化训练数据集所有标签类别的联合预测概率。
